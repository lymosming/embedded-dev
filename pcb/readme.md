1. 新建项目：新建 原理图库 / PCB库 / 原理图 / PCB

2. 原理图库可以画器件，用于在原理图里画原理图

3. 原理图绘制：
    1）放置所有元器件
    2）划分区块
    3）连接导线
    4）放置网络

4. 元器件封装添加
    1）自己画封装 2D
    2）引用第三方封装库的封装 / 直接复制别的项目的封装
    3）丝印层：Top Overlay  
    4）画3D模型 / 导入第三方3D模型

5. PCB
    1）白色圆圈是短路
    2）工具=》设计规则检查 去关闭一些规则保留电气规则可以解决部分绿色报错
    3）设定板框大小：全选器件，按框型摆放，定义稍微大一点接口
    4）在机械层 Mechanical 1画板框
    5）四层版比2层板简单，2层板走线设计可能更复杂，但是四层板更贵
    6）正片层：画线即为铜； 负片层：画线为去除铜
    7）左边原理图，右边pcb，然后框选原理图模块，右边按矩形区域摆放器件，就可以快速选出整个模块的器件
    8）创建PWR网络类，添加VCC GND等电源网络进去，可设置整体隐藏他们的飞线，只留下信号线，可以更直观布局信号线
    9）布局：先大模块再小模块，按信号流向摆放，根据使用习惯评估位置合理性
    10）减小器件号标注字体：选中一个，查找相似 =》 设置高度10mil 宽度2mil
    11）器件号标注居中：CTRL+ A全选 =》
